Tutorials suite covering the \href{https://pressio.github.io/pressio/html/index.html}{\texttt{ pressio C++ library}}. \hypertarget{index_autotoc_md1}{}\doxysection{Build and Run}\label{index_autotoc_md1}


All you need CMake $>$ 3.\+18.\+0 and a C++14 compliant compiler. 


\begin{DoxyCode}{0}
\DoxyCodeLine{git clone -\/-\/recursive git@github.com:Pressio/pressio-\/tutorials.git}
\DoxyCodeLine{export CXX=<path-\/to-\/your-\/CXX-\/compiler>}
\DoxyCodeLine{cd pressio-\/tutorials \&\& mkdir build \&\& cd build}
\DoxyCodeLine{cmake -\/DCMAKE\_BUILD\_TYPE=Release ..}
\DoxyCodeLine{make -\/j4}

\end{DoxyCode}
